ABOUT THE PROJECT

Weather Dashboard is a futuristic web application designed to provide accurate real-time weather forecasts for any city in the world. Using the OpenWeather API, the application fetches live weather information and presents it through a modern glassmorphism interface enhanced with smooth animations and dynamic effects. The project was built to combine clean UI design with practical functionality while demonstrating frontend development skills using HTML, CSS and JavaScript.
